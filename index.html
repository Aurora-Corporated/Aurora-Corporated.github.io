Chapter I:只有神知道的地方

发表于 2020-08-15

只有神知道的地方

​ 与那遥远的天空相连的一段，是命运。

“熵的增加…是什么时候开始的呢？“她对自己说道。

自从他们离开之后，地球似乎变得平静了下来。日复一日的学习，工作，不断地被重复着的生活，到了最后，这些事情终究还是没有意义，宇宙终究要被自己的存在所压垮，而且，神不在这里了。她苦笑了两声，继续躺在床上，盯着空洞的天花板。

”神啊…你为什么要把我抛弃在这里?“ 眼泪流了下来。

即使自己再怎么祈祷，那位神灵也不会听见的吧。

…

她睡着了。

记忆

​ 雨停了。

这样做下去…还有什么意义呢？别人都不可能相信我的…到底是我自己在妄想，还是…?

晓美焰拿起了手枪，指向了自己的太阳穴。

自然而然的，流下了眼泪。

“到最后还是没有结果的等待，那不就是没有意义的吗..? 我之前…我之前所做的一切，都白费了吗？

神啊，你为什么要对我这么残酷？”

然而，枪声并没有响起。她注意到了头上凌乱耷拉着的红色的发箍。

这，就是答案。

它证明了一切都是实实在在的发生过的。

焰顾不得多梳妆打扮，潦草的套上了学生服，前去寻找线索。

第一站，就是那里了。

黄昏映照着公寓楼。焰沿着楼梯，向上行进。“自从那家伙离开之后，这个地方应该也没有什么人来过吧？“

门是虚掩着的，轻轻一推就能推开。房间内部十分整齐，几乎没有什么改变。

唯一变化的就是灰尘的数量。

“熵增…”

徘徊之时，她注意到了一件不同寻常的东西，一件本不应该在这里的东西。是那个笔记本。它静静的躺在茶几旁的柜子上，是巴麻美离开时的模样。

“鹿目圆香”

扉页告诉了主人的名字。

焰终于忍受不住那交织的情感，哭了出来。

还好。还好自己没有完全被神抛弃掉。

如何用github创建一个免费的博客

发表于 2020-08-13 更新于 2020-08-14

博客の创建

这一篇教程将会教你如何在Github上面创建(baipiao)一个基于hexo的博客。

一、准备

在这一篇教程里，你要用到的资源有:

  • 一台电脑 废话

  • Git-scm

  • Node.js运行环境

  • 一个Github账号

关于Node.js和Git-scm的安装过程我不会多做描述。如果你已经有Github账号请跳过第二章。

二、注册Github账号

请访问这个界面:https://github.com

这既是github的主页。在右边你可以看到三个灰色的文本输入框和一个绿色的按钮。

标注为Username的文本框是用户名的意思，请想好你的用户名，然后填写进去。（最好是易于记忆的用户名）标注为Email的文本框是电子邮箱的意思。这里可以填写任何你拥有的邮箱，如果不会注册请百度”邮箱的注册”。标注为Password的文本框是密码的意思。在这里请填写只有你一个人知道的易于记忆的密码。

这三个项目都填写完成之后，请点击绿色的按钮，之后你将会来到验证界面。

验证界面中，应该有一个向左的箭头和向右的箭头，中间还有一个动物的图片。指向左边的箭头将会把动物图片往左边转，指向右边的箭头将会把动物图片往右边转。你的目的就是把图片转到水平的位置，让动物的位置看起来正常一点，之后点击提交。

在做完了验证码之后，请打开你的邮箱，然后选中确认邮件，打开，点击确认地址。填写完一些你感兴趣的项目之后再次访问https://github.com,你就会发现你来到了你的GitHub主页。

三、创建库

点击左边项目栏中绿色的按钮，创建一个新的Repository - 仓库，你就会来到这个界面。

你需要改动的地方有:

  • Repository Name
  • Public/Private

Repository Name中请严格按照规则填写: 你的用户名.github.io。假设你的用户名是test，那么Repository Name就应该是 test.github.io。

同时，确保Public和Private两个选项中选择的是Public。完成之后选择Create Repository。

如果你看到了这个界面，那么恭喜你，你成功了！！！既然你已经安装好了Git,那么下一步就是生成SSH密钥对，然后上传我们的博客。

四、生成密钥对

进入桌面，右键。你会看到Open Git Bash Here的选项，点击它，之后在控制台里面输入ssh-keygen。

显示出类似以下这个界面的窗口时，恭喜你，你又成功了。

之后用笔记本打开 C:\Users\你的Windows用户名（如果你用的是微软在线账户的话是你邮箱的前五位）\ .ssh\id_rsa.pub，复制里面的内容。

回到Github首页，点击你的头像，在跳出来的页面里选择SSH and GPG keys。点击绿色的New SSH key按钮，之后在Title里面写上你能够记住的标识，在Key的栏位里面把刚才复制的东西粘贴进去，点击绿色的Add SSH key。
